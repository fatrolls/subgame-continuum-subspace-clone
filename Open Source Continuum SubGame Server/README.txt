You will require all the files in this directory to be present to run a server.

The UPDATE.EXE autoupdates the client if it is out of date. If you are running a server then check our server board for new updates.

Important! The BILLING server password is auto generated so your game server will not match to begin with! Change the billing pw in the server.ini file to duplicate the pw the subill.ini
