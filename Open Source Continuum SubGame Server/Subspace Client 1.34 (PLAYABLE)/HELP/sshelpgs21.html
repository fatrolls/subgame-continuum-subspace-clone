Operating System

Microsoft Windows 95 and Micrsoft Windows NT4.0 are the only operating systems that can run SubSpace.

One of the most common questions we get in the SubSpace email-box is "When is the Macintosh version coming out?". Sorry to say that it will most likely never happen.

SubSpace started out as an Internet technology test and was never meant to be a real product. As such, we designed it to work with Microsoft's DirectX technology. The only operating systems that support DirectX are Microsoft Windows 95 and Windows NT4.0. Unless there are dramatic changes in Mac OS (or UNIX or LINUX etc.) there will not be a version of SubSpace for other platforms.
