Map Editor(s)

Check the Utilities Page on our website (subspace.vie.com) for user made map editors.

To use the "Built-in" editor: (all picture references are for the standard tileset)

**note: Map is saved "on the fly" (or on exit, haven't tested this yet) as local.lvl

**note: To edit an existing level, copy and rename current map (you may have to do some timestamp checking) to levelname.lvl where levelname is a name of your choice.
To start with a blank map, delete local.lvl or if you want to use a different tileset and you have a proper BMP graphic, you can just rename the BMP file to local.lvl.

First, select PLAY OFFLINE as a zone then hold down SHIFT and CTRL when selecting ENTER ZONE to enter editor. --It doesn't matter if you are connected to the internet or not.

Enter brings up a RADAR type display so you can see what you are doing.
Arrow keys move the locator on your Map.
Shift+Arrow keys move the locator on the Tile Box.

Spacebar - Places selected tile or deletes if already on a tiled section.
Delete - Deletes tiled section (like I had to explain but you just never know).
Ctrl - Places selected tile. (use with arrow keys to draw straight or diagonal lines quickly).

(Capital) D - Space Station (6x6 tiles)
(Capital) E - Gravity Warp (4x4 tiles)
(Capital) A or C - Small Asteroid (1x1 tiles)
(Capital) B - Medium Asteroid (2x2 tiles)
[Note: There are no Large Asteroids anymore.]

SELECT and ERASE/COPYnPASTE
1. (Capital) S - Start SELECT
2. Use arrows to size box
3. (Capital) S[again] - Set size of box (will copy what box surrounds)
4a. (Capital) E to Erase contents of box
4b. (Capital) P to Paste what was copied in #3

Escape to exit and save
