View Menu

News- This menu option (active only when connected to a Zone) views the latest news about the game.

Message Hot Keys- This menu option allows you to set up to 8 (16 if you have the 'Distinguish left/right' option checked) text macros and 2 sound files for use in the game.

Options- This menu option brings up a dialogue box that allows you to change the way SubSpace looks when you are in the game.

Weapon Gauge: Standard shows full graphical representations of all weapons and devices, while Text only displays the text description and Off turns all representations off.

Status Gauge: Standard shows full graphical representations of your ship’s status including Energy level, frequency, flags (personal and team) and your bounty, while Text only displays the text description and Off turns all representations off.

Energy Bar: Standard shows a full graphical representation of a double-sided energy bar, while Text only displays a minimal graphical representation and Off turns it completely off.

Radar: Small shows a 4 sector by 4 sector graphical representation of a double-sided energy bar, while Large displays the same information but the blips are a little more accurate and Off turns it completely off.

Font: Short displays normal text, while Tall displays a taller font and Legible displays an older style font that shall remain nameless (any SINCLAIR users out there?).

Sound: On means all game sounds will be active, while Engines Off will turn off engine sounds and background noise and Off turns all sounds off.

Miscellaneous: Check boxes (when checked)Show Clock-shows current server time above Radar; Show Backgrounds-turns on background bitmap images that appear as planets, star clusters, etc.; Show Stars-turns on starfield background; Gamma Correction-makes screen brighter; Shaded Boxes-puts gray background behind status graphics; Opaque Message-does not allow stars to show up behind messages; Wave Messages-allows you to hear wave files played by other users; Block Obscenity-blocks text messages with known obscene words (not configurable); Message Ticker Help-turns on helpful hints while playing the game; Message Alarm-turns on audible signal that alerts you when you get a provate message; De-dupe Messages-forces client to check for and remove duplicate messages; Ignore Public Macros-ignores macros sent on the public channel; 3d Sound Effects-will use 'surround-type' sound (if available); Message Entry Box-when checked, will keep a dialog box active if you ALT-TAB out of the game; Detailed Explosions-turns on graphical explosion of ship (bits fly out everywhere); Rolling Ship-turns on and off the "Ship Roll" feature in the game; Joystick-turns on ability to use joystick; Software Emulation-disables DirectX hardware acceleration; Translucent Effects-allows some game items (like explosions) to have levels of transparency; Show Ping Times-At the connect screen, will show numbers instead of graphics; Skip Main Menu-will skip Main Menu when starting the game; Visible Thrust-toggles engine flame on and off; Mulit-fire Start On-initial starting toggle for Multi-fire Guns; Bomb Trails-toggle for bomb trail graphic; Bullet Trails-toggle for bullet trail graphic; Shrapnel Trails-toggle for shrapnel trail graphic;

Message Lines-Number of message lines displayed in the game before old messages tart scrolling off the top.

Target Bounty-Sets bounty level for radar[level when dark blue ships turn light blue (self-imposed honor system)].
