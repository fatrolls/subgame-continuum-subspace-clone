Help Menu

Message Ticker Help- This menu option displays if you have the Message Ticker Help option checked in the VIEW/Options dialogue box (this is for display purposes only).

Tutorial- This menu option starts a brief offline interactive demo to help you get used to the basic commands.

Quick Help Screen- This menu option brings up a screen with all the game elements that display a description when you move your mouse pointer over the options.

Contents- This menu option accesses this help file.

About- This menu option displays SubSpace Team information.
